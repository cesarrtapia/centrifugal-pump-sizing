## 🧩 Overview

This project is an engineering-oriented calculation tool for the **selection and sizing of centrifugal pumps** in industrial systems.

It includes:

* Flow rate calculations
* Total Dynamic Head (TDH)
* Friction losses (Darcy-Weisbach)
* Minor losses (fittings & valves)
* Hydraulic and shaft power
* NPSH verification

---

## 🎯 Objective

Provide a clear, structured, and reusable engineering methodology for pump selection.

---

## 🏗️ Features

* 📊 System head calculation (TDH)
* 🌊 Flow analysis
* ⚡ Power estimation
* 🚫 Cavitation check (NPSH)
* 🔧 Engineering-ready structure
